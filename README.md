# Jane Distribuidora - Projeto Final

Projeto desenvolvido para a disciplina do 2º período de Sistemas para Internet.

## Objetivo

Criar uma página institucional/portfólio para uma empresa real, usando HTML, CSS, JavaScript e Bootstrap.

## Empresa escolhida

Jane Distribuidora, com proposta visual inspirada em sites de distribuidoras de bebidas, porém com redesign mais moderno, institucional, comercial e responsivo.

## Dados públicos usados no conteúdo

Foram utilizados dados públicos encontrados na presença online da empresa e em resultados públicos de busca:

- Perfil: `@janedistribuidora`
- Localização pública: Palmas, TO
- Segmentos destacados: Bebidas, Alimentos e Ferragens
- Comunicação da marca: logística ágil em Palmas e região
- Forma de pedido: balcão ou vendedores
- Público mencionado em publicações: pizzarias, padarias, mercados, lanchonetes e empresas
- Produtos/marcas citados em publicações públicas: Pitchula e Grelhaditos
- WhatsApp divulgado em publicação: `(63) 9219-1159`

## Tecnologias usadas

- HTML5 semântico
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome
- Google Fonts
- Imagens externas carregadas por URL

## Funcionalidades implementadas

- Menu responsivo com Bootstrap
- Header/hero institucional
- Seção Sobre
- Seção de presença comercial
- Seção Categorias
- Catálogo com filtro por categoria usando JavaScript
- Seção de estrutura/processo comercial
- Portfólio com imagens e modal Bootstrap
- Formulário de contato integrado ao WhatsApp
- Botão flutuante de WhatsApp
- Botão voltar ao topo
- Layout responsivo para celular, tablet e desktop

## Observação

O projeto é acadêmico e sem fins comerciais. As imagens externas precisam de internet para carregar no site publicado. Marcas citadas pertencem aos seus respectivos proprietários.
